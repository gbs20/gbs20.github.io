Markdown 基本语法

2020-01-13
Language

Markdown是一种纯文本格式的标记语言。通过简单的标记语法，它可以使普通文本内容具有一定的格式。

  1. 标题

    H1

    H2

    H3

  2. 缩进 使用 三个缩进空格 表示缩进。

  3. 外链接

    使用 [描述](链接地址) 为文字增加外链接。

    示例：

    这是去往 本人博客 的链接。

  4. 插入图像

    使用 ![描述](图片链接地址) 插入图像。

    示例：

  5. 换行与段落

    在markdown语言中,唯一决定两行文字是否是段落的,就在于这两行文字之间是否有空行.

  6. 删除线

    使用 ~~ 表示删除线。

    这是一段错误的文本。

  8. 表格

  1. 文字引用

    使用 > 表示文字引用。

    示例：

野火烧不尽，春风吹又生。

  1. 行内代码块

    使用 代码 表示行内代码块。

    示例：

    让我们聊聊 html。

  2. 代码块

使用 四个缩进空格 表示代码块。

示例：

这是一个代码块，此行左侧有四个不可见的空格。
